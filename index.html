HTML Cheatsheet
  1. HTML Boilerplate: This is the basic html boilerplate code 
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
  2. OL Tag: This is the basic ordered list 
    
                    <ol>
                        <li> </li> 
                        <li> </li> 
                        <li> </li> 
                    </ol>
  3. Paragraph tag: This is the paragraph tag 
    <p> </p>
  4. Div tag: This is the div tag 
    <div> </div>
  5. Var declaration: This is variable declaration using var 
    var i = 45;
  6. Let declaration: This is variable declaration using let 
    let i = "Hello!";
  7. Constant declaration: This is variable declaration using const 
    const i = false;
